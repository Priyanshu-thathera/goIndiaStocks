## Getting Started

Setup:

1. Clone the repository: First clone the code from my repository
git clone https://github.com/Priyanshu-thathera/goIndiaStocks.git

2. Navigate to the project directory:
cd goIndiaStocks

3. Install dependencies:
npm install
# or
yarn install

5. Running the project:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
